# ⚖️ Legal Service Nepal

A web platform providing comprehensive legal service information and resources for Nepal. Built with modern web technologies and deployed live on Vercel.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white)
![Vercel](https://img.shields.io/badge/Deployed_on-Vercel-000?style=flat-square&logo=vercel&logoColor=white)

🔗 **Live Demo**: [legal-service-nepal.vercel.app](https://legal-service-nepal.vercel.app)

---

## 📋 Features

- 📖 Comprehensive legal service directory for Nepal
- 🔍 Easy navigation to find legal resources
- 📱 Responsive design for mobile and desktop
- ⚡ Fast loading with Vercel deployment

---

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| HTML5 | Page structure and content |
| CSS3 | Styling and responsive layout |
| JavaScript | Interactive functionality |
| JSX/React | Component-based UI |
| Vercel | Hosting and deployment |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/aditya-mandal12/legal-service-nepal.git
cd legal-service-nepal

# Open in browser
open index.html
```
